# x402 Pay-Per-Scan Tests (REMOVED)

**Status:** REMOVED in v0.47.0 (March 14, 2026)
**Reason:** x402 USDC micropayment system removed alongside wallet authentication. Platform uses Stripe-only payments.
**See:** [Security Audit - Wallet Removal](../audit/security-audit-2026-03-14.md)

x402 pay-per-scan (USDC on Base blockchain) has been removed from the platform. All payments are processed through Stripe (card-based subscriptions and one-time purchases).
